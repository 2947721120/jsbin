Help

Tips

  • 点击HTML或JavaScript隐藏JavaScript面板标签（这是存储你的喜好）.
  • 只显示HTML面板使用# HTML在URL的结尾，如http://jsbin.com/#html
  • “使用模版” 存储当前的代码作为默认的代码你看当你打开一个新窗口JS仓。
  • 刷新窗口不会丢失您的更改。
  • 使用 console.log 显示调试信息。如果它是存在的，那么记录那里去，否则会在控制台窗口中显示一个自定义的。
  • 只查看JavaScript添加/js到这个url, i.e. http://jsbin.com/ujayi/js

键盘导航

  • ctrl + 1查看源代码
  • ctrl + 2 查看预览
  • ctrl + ← JavaScript的焦点
  • ctrl + → 焦点HTML

常见问题

当我更改url到不同的JS斌，它仍然有旧代码

点击恢复 revert 按钮，它会根据您请求的URL重新从数据库的代码。.

我可以运行在一个JSBin要点？

当然你可以 - 只是保存 gist 在Github上，要么和使用 gist to JS Bin书签，或更改网址 http://gist.github.com/271333到 http://jsbin.com/gist/271333

我没有插入JavaScript和它仍然运行

当你卡到输出面板，JS斌自动插入您的飞行JavaScript，因此你不必担心。

我需要我的JavaScript插在特定位置

无论是插入的JavaScript直接到HTML面板，也可以使用特殊%code%命令。 加<script>%code%</script> 你的HTML和JavaScript面板的内容将在该点插入.

我该如何重新运行或重新渲染代码

只要按一下‘Preview’ 按钮，你的整个代码将重新运行.

JSBin可以响应Ajax请求？

是。见演示阿贾克斯调试视频（在导航的顶部）。或者，它’能够像删除HTML输出，节省了代码，然后通过一个新片段请求URL一样简单。 JSBin将适当响应Ajax请求.
